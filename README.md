# Pig Game

A fun dice-rolling game where players compete to reach a target score first.

## How to Play

Roll the dice to earn points, but be careful! If you roll a 1, you lose all points for that round. Choose to "Hold" to bank your points and pass the dice to your opponent.

## Features

- Two-player gameplay
- Simple and intuitive interface
- Score tracking
- Responsive design

## Getting Started

1. Open `index.html` in your web browser
2. Click "New Game" to start
3. Players take turns rolling the dice
4. Roll the dice to accumulate points, or hold to save your score
5. First player to reach 100 points wins!
